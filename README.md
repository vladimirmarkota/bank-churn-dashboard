# Bank Customer Churn Dashboard

This is my first **Power BI dashboard** analyzing **bank customer churn**.  
The project focuses on understanding customer behavior and visualizing key metrics.

## Project Overview
- Analyze churn rate based on demographics (Age, Gender, Geography)
- Measure financial indicators (Average Balance, Number of Products)
- Explore behavioral signals (Tenure, Product usage)
- Create actionable insights for retention strategies

## Technologies
- **Python / Pandas** – Data cleaning and preparation
- **Power BI Desktop** – Dashboard creation and visualization
- **Kaggle Dataset** – [Bank Customer Churn](https://www.kaggle.com/datasets/radheshyamkollipara/bank-customer-churn)

## Folder Structure
/data # Cleaned CSV data
/notebooks # Python notebooks for data preparation
/dashboard # Power BI dashboard file (.pbix)
/images # Screenshot of the dashboard
/README.md # This file

## Key Metrics & Visuals
- **Churn Rate** – overall and by segment
- **Total Customers**  
- **Average Balance & Products per Customer**  
- **Churn by Age, Gender, Geography**  
- **Behavioral Risk Indicators** – Balance, Products, Tenure

## How to Use
1. Open `dashboard/bank_churn_dashboard.pbix` in Power BI Desktop
2. Explore different visuals and filters
3. Check Python notebook (`notebooks/data_cleaning.ipynb`) for data preparation steps
